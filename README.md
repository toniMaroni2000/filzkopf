# Filzkopf

Ein einfacher, moderner One-Page-Auftritt für **Filzkopf** von Liz Keller.

Die Website besteht nur aus statischem HTML, CSS und JavaScript und ist damit direkt für GitHub Pages geeignet.

## Dateien

- `index.html` - Inhalt und Seitenstruktur
- `styles.css` - Gestaltung, Layout und Responsive Design
- `script.js` - Mobile Navigation und Jahreszahl im Footer
- `assets/` - Bilder und Platzhalter für Galerie und Hero

## Bilder ersetzen

Die Galerie verwendet Platzhalterbilder aus `assets/`. Eigene Fotos können dort abgelegt und in `index.html` bei den entsprechenden `<img>`-Tags eingetragen werden.

## Deployment auf GitHub Pages

1. Ein neues GitHub Repository erstellen.
2. Diese Dateien in das Repository pushen.
3. In GitHub öffnen: **Settings → Pages → Deploy from branch → main → root**.
4. Optional eine eigene Domain wie `filzkopf.ch` einrichten.

Nach dem Speichern der Pages-Einstellungen veröffentlicht GitHub die Website automatisch.
